What is APhysics Tools

APhysics Tools is an open source collection of simple to use tools for the AP Physics 1 course, it houses a variety of tools, such as a 2D Vector addition calculator, some other thing, another thing, and more! Additionally, there is documentation on all the used equations, when those equations can be used, and different ways to approach a given problem!

About Us

APhysics Tools is being developed, and maintained by two current AP Physics 1 students. This project will be continuously changing, with the addition of new features such as new calculators, simulations, and documentations.
